# small-ledger
家有小账本 - 个人生活管理微信小程序

## 项目概述
一个集任务管理(Todo List)和个人记账功能于一体的微信小程序，帮助用户高效管理日常任务和财务状况。

## 核心功能

### 任务管理系统 (Todo List)
- **基础功能**：支持创建、修改、删除任务，以及更新任务状态（待办、进行中、已完成等）
- **时间维度分类**：按照每周、每月、每年的时间周期组织和查看任务
- **四象限分类法**：根据任务的重要性和紧急程度，将任务分为四类
  - 重要且紧急
  - 重要不紧急
  - 不重要但紧急
  - 不重要不紧急

### 记账功能
- **收支记录**：记录每日的收入和支出，支持分类（如餐饮、交通、购物等）
- **攒钱目标设定**：设定不同时间周期的攒钱目标
  - 月度目标
  - 季度目标（三个月）
  - 半年目标
  - 年度目标
- **目标进度追踪**：可视化展示攒钱目标的完成进度

## 技术栈
- 前端框架：Vue 3 + TypeScript
- 开发工具：Taro 4
- 状态管理：Pinia
- 样式处理：Tailwind CSS + SCSS
- 组件库：Nutui
- 图标库：@nutui/icons-vue-taro
- 目标平台：微信小程序

### 设计理念
- **简约美学**：采用简洁的设计语言，减少视觉干扰，突出核心功能
- **分层视觉**：通过卡片、阴影和空间布局创造视觉层次感，提升界面深度
- **直观交互**：提供即时反馈和明确的操作指引，降低用户学习成本
- **一致体验**：保持应用内的设计风格一致性，确保用户在不同功能间无缝切换

### 界面特色
- **现代化配色方案**：以紫色系为主色调，搭配中性色，营造温馨专业的视觉氛围
- **优雅的动画效果**：平滑的过渡动画和微交互，提升用户操作体验
- **清晰的数据可视化**：使用图表和进度条直观展示财务和任务数据

### 交互优化
- **即时反馈**：操作按钮点击时有明显的视觉反馈，提升交互确定性
- **智能表单设计**：优化输入框焦点状态和表单验证提示，提升数据录入效率
- **直观的操作提示**：使用图标和文字组合，明确指示功能区域和操作目标
- **空状态处理**：精心设计的空状态界面，引导用户开始使用各项功能

## 功能模块
1. **待办事项管理系统**
   - 实现基础的任务管理功能：创建、编辑、删除、标记完成
   - 支持任务优先级设置（高、中、低）
   - 提供双维度任务分类视图：
     * 时间周期维度：按周、月、年筛选和展示任务
     * 重要性维度：采用四象限法则（重要且紧急、重要不紧急、不重要但紧急、不重要不紧急）
   - 支持列表视图与四象限视图的灵活切换
   - 所有数据采用本地存储持久化
   
   ### 新建任务页面
   - **完整的任务创建表单**：包含任务名称、描述、时间周期和优先级设置
   - **智能表单验证**：实时验证输入内容，提供清晰的错误提示
   - **直观的优先级选择**：采用四象限可视化布局，方便用户快速选择任务优先级
   - **响应式设计**：适配不同尺寸的设备屏幕
   - **友好的操作反馈**：任务创建成功后显示提示信息
   
   ### 编辑任务页面
   - **数据预填充**：加载现有任务数据，自动填充表单
   - **完整的编辑功能**：支持修改任务名称、描述、时间周期和优先级
   - **表单验证**：与新建任务页面保持一致的验证逻辑
   - **操作反馈**：任务更新成功后显示提示信息
   - **数据一致性**：确保修改后的数据与任务列表保持同步

2. **记账功能模块**
   - 支持记录日常收入和支出
   - 提供丰富的预设收支类别（如餐饮、交通、购物、工资等）
   - 实现月度收支统计和概览展示，包含收入支出趋势对比
   - 优化的记账表单，提供直观的类别选择和金额输入体验
   - 提供记账记录的查询、编辑和删除功能

3. **攒钱目标设定模块**
   - 支持创建不同时间周期的攒钱计划（月度、季度、半年、年度）
   - 实时显示目标进度和完成百分比，采用渐变进度条直观展示
   - 智能计算每日/每周所需贡献金额
   - 优化的目标创建和编辑界面，提供清晰的期限选择和金额输入
   - 提供目标历史记录和完成情况回顾，支持查看历史目标详情和进度


## 下期功能

### 1. 登录页添加邮箱验证
- **用户注册**：用户通过注册页面填写个人信息（如用户名、邮箱、密码）进行账号创建
- **邮箱验证**：注册成功后，用户需要通过邮箱接收的验证码进行验证，确保邮箱的真实性
- **登录功能**：验证通过后，用户可以使用邮箱和密码登录应用
- **密码找回**：如果用户忘记密码，提供密码找回功能，通过邮箱发送重置密码的验证码
### 2. 账单页添加右下角添加按钮
- **显示添加账单弹窗**：弹窗中显示h5的路径
- **添加目标开支**：用户可以添加目标开支，最终以饼图展示目标进度，包括已完成和未完成部分，如果超支目标，以红色展示
- **柱状图展示每周，每月的开支**：用户可以在账单页查看每周和每月的开支情况，以柱状图形式展示

### 3. 任务添加关联账单（如：人情往来）
- **任务创建**：用户在待办事项管理系统中创建新任务时，可选关联已有的账单记录
- **任务与账单关联关系**：用户可以在任务详情页查看关联的账单记录，也可以在账单详情页查看关联的任务
### 4. 小程序打包自动化
- **自动构建**：在代码提交后自动触发构建流程, 并将构建好的小程序代码上传到微信小程序平台
### 5. 前后端改为monorepo(方便ai拿到统一上下文)
- **统一代码仓库**：将前端和后端代码统一存储在一个代码仓库中，方便管理和协作
- **共享代码模块**：前端和后端可以共享一些通用的代码模块
- **独立部署**：前端和后端可以独立部署，互不干扰
